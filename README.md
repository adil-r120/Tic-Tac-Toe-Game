# Tic Tac Toe Game

A modern, responsive Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- **Game Modes**:
  - Player vs Player
  - Player vs Computer (with difficulty levels: Easy and Hard)

- **Design**:
  - Beautiful modern UI with glassmorphism effects
  - Smooth animations and transitions
  - Responsive design that works on all devices
  - Light and Dark mode toggle

- **Gameplay**:
  - Intuitive gameplay with visual feedback
  - Win detection with highlighted winning combinations
  - Draw detection
  - Motivational quotes when a player wins

## How to Play

1. Choose your game mode (Player vs Player or Player vs Computer)
2. If playing against the computer, select difficulty level (Easy or Hard)
3. Take turns placing X's and O's on the board
4. The first player to get 3 in a row (horizontally, vertically, or diagonally) wins
5. Click "Reset Game" to start a new game

## Technologies Used

- HTML5
- CSS3 (with modern features like Flexbox and CSS Variables)
- JavaScript (ES6+)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/adil-r120/Tic-Tac-Toe-Game.git
   ```

2. Open `tic_tac_toe.html` in your browser
